# frontend/ — 지도 UI (D)

지도 위에 출발점·우선순위 마커·동선 폴리라인을 그리는 화면.
> ⚠️ 아직 코드 미작성. **프론트엔드 선택(Q3)이 확정되면** 구현 시작.

## 결정 대기 (PROJECT_HANDOFF.md Q3)

| 옵션 | 장점 | 단점 |
|---|---|---|
| **Streamlit** (+folium) | 파이썬만, 빠른 시작, 배포 쉬움 | 인터랙션 한계 |
| **React + Leaflet** | 웹 정석, 인터랙션 자유, 학습 깊음 | 진입장벽 ↑ |

> 둘 다 **Leaflet + OpenStreetMap 타일**(무료·무키)을 지도 기본값으로 사용.

## 화면 요소 (예정)

- 지도 + 출발점 A 선택(클릭)
- 컨트롤: 예산(30분/1h/2h/3h), 신호 슬라이더(v1.2), 인구 필터(v1.3)
- **우선순위 마커**: 도달 가능 후보를 순위로 표시
- **동선 폴리라인**: A→B→C… 실제 보행 경로(v1.1)
- 인접 구로 넘어감 표시(D7)

## v1.0 목표

출발점에서 도보 예산 안에 닿는 스팟을 지도에 마커로 띄우는 **단일 스텝** 화면("도보 n분 스팟지도").
