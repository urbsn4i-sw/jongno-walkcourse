# models/ — 랭커 학습/추론 & 벤치마크 (레이어 C의 NN 부분)

추천 점수 중 **학습 랭커(NN)** 담당. 도달성/시간 계산은 여기 없음(그건 `backend/`의 결정론 엔진).
> ⚠️ 아직 코드 미작성 — 주로 **v1.4**에서 구현. 무거운 학습은 Colab.

## 역할

- **NN은 후보 랭킹만** 한다 (시간/도달성 아님).
- Foursquare 공개 체크인으로 **next-POI 랭커**를 학습·평가 → **Recall@k / NDCG** 보고.
  → "진짜 ML 했다"의 정량 물증.

## 서사 (베이스라인 → 딥러닝)

| 단계 | 모델 |
|---|---|
| 베이스라인 | KNN · 로지스틱 · 의사결정나무 · SVM · 앙상블 |
| 딥러닝 | RNN / Transformer / GNN |
| 표현학습 | DBSCAN 군집, PCA/UMAP 차원축소로 POI 임베딩 |

## 서울 적용 (데모)

- 학습된 구조를 종로 POI에 적용.
- 전이 신호: 데이터랩 "관광지별 연관 관광지" + 부족분은 합성 궤적.
- **정직한 한계:** Foursquare→종로 도메인 갭. 사용자에게 보이는 순위는 해석가능 신호(언급량·군집·연관관광지)가 주로 끌고, 학습랭커는 벤치마크 증거 역할.

## 예정 구조 (제안)

| 경로 | 역할 |
|---|---|
| `train.py` | 랭커 학습 |
| `evaluate.py` | Recall@k / NDCG 평가 |
| `infer.py` | 종로 POI 추론(랭킹) |
| `checkpoints/` | 학습 가중치 (git 추적 안 함) |
